# Credit Card Segmentation

This project performs customer segmentation based on credit card data using machine learning techniques. It is intended for exploratory analysis and modeling to identify distinct customer groups for marketing, risk assessment, or product recommendations.

## Features

- Data cleaning and preprocessing of credit card transaction data
- Exploratory data analysis (EDA) with visualizations
- Clustering algorithms (e.g., GMM, KMeans)
- Model evaluation and interpretation
- Interactive Jupyter Notebook for experimentation

## Repository Structure

```
Credit Card Segmentation/
├── Credit Card Segmentation.ipynb   # Main notebook with analysis and modeling
├── data/                            # Dataset directory
├── requirements.txt                 # Python dependencies
└── README.md                        # Project documentation
```

## Getting Started

### Prerequisites

- Python 3.8 or later
- Jupyter Notebook

### Installation

1. Clone the repository:
```bash
git clone <repo-url>
cd "Credit Card Segmentation"
```

2. (Optional) Create a virtual environment:
```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

### Usage

1. Launch Jupyter Notebook:
```bash
jupyter notebook
```

2. Open `Credit Card Segmentation.ipynb` and run through the cells.

3. Modify or extend clustering experiments as needed.
